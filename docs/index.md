# FlagControl SDKs

Welcome to the official documentation for FlagControl SDKs.

## Features

- **Type-Safe**: End-to-end type safety for your feature flags.
- **Performance**: Local evaluation for Node.js, optimized remote evaluation for React.
- **Flexibility**: Support for complex rules, lists, and dynamic contexts.
- **Privacy**: Hashed list evaluation to protect user privacy.
